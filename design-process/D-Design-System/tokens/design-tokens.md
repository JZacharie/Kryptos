| Token | Hex | Usage |
|-------|-----|-------|
| `base-dark` | `#0A0E14` | Background / Main surface |
| `surface-glass` | `rgba(20, 24, 32, 0.7)` | Glassmorphic containers |
| `accent-primary` | `#00E5FF` | Primary actions / Healthy status |
| `accent-secondary` | `#FF00E5` | Interactive highlights |

| `text-dim` | `#94A3B8` | Low emphasis text |
